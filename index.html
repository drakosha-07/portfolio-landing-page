You don't have to be great to start, but you have to start to be great." - Zig Ziglar

I am student of the coding bootcamp Epicodus in Portland, Oregon. In the introdcutory course that lasted three weeks we learned how to create and manage a Gihub repository, use Terminal and Visual Stuido Code, create webpages using Markdown and HTML, and style those webpages with the help of CSS and Bootstrap. I never thought I could create a website myself, but it's actually not that hard. It's pretty fun too!

This webpage examplifies everything I've learned so far, like the use of different tags, divs, classes, styles. You can't see the full set of requirements because it's published on the student portal, but you can see the code for this website in its Github repository.
